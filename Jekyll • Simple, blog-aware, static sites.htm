快速入门

~ $ gem install jekyll

~ $ jekyll new my-awesome-site

~ $ cd my-awesome-site

~/my-awesome-site $ jekyll serve

# => 现在在浏览器输入 http://localhost:4000

在GitHub Pages制作免费的域名博客

厌倦了一些费时费钱的博客? GitHub Pages 支持 Jekyll, 所以你可以制作一个属于你自己的免费的博客在 GitHub— 可以自定义域名和其它 自己喜欢的.
